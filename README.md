# Projeto-Mortal-Kombat
**Mortal Kombat: A Batalha do Código!**
⚔️ Sobre o Projeto

Prepare-se para um Fatality de código!  Este projeto web te leva para o universo de Mortal Kombat, mas dessa vez, a luta é para encontrar seus personagens favoritos. Com uma interface simples e intuitiva, você pode pesquisar por qualquer lutador e descobrir suas habilidades e história.

️ Tecnologias Utilizadas

HTML: A espinha dorsal da nossa página, responsável pela estrutura.
CSS: Deixando tudo bonitinho e responsivo para qualquer tela.
JavaScript: A mágica por trás da pesquisa! Fazendo tudo funcionar direitinho.
** Como Funciona?**

Digite o nome do personagem: No campo de pesquisa, digite o nome do lutador que você quer encontrar.
Aperte o botão "Pesquisar": E pronto! A aplicação vai vasculhar nossa base de dados e te mostrar todos os resultados relevantes.
Explore os resultados: Clique nos links para saber mais sobre cada personagem.
‍ Para os Curiosos

dados.js: Esse arquivo contém todas as informações sobre os personagens, como nome, descrição e imagem. É como a torre dos magos, guardando todos os segredos.
index.js: Aqui que a magia acontece! É nesse arquivo que a função pesquisar() faz todo o trabalho de buscar os personagens na base de dados e exibir os resultados na tela.
⚠️ Atenção, Noob Saibot!

Case-sensitive: A pesquisa não diferencia maiúsculas de minúsculas, então pode digitar como quiser!
Resultados dinâmicos: A cada nova pesquisa, os resultados são atualizados na tela, sem precisar recarregar a página.
** Próximos Rounds**

Mais personagens: Vamos continuar adicionando mais lutadores para deixar a batalha ainda mais épica!
Filtros: Quer encontrar só os personagens de um determinado reino? Em breve teremos filtros para isso!
Modos de jogo: Quem sabe no futuro a gente não cria um modo de jogo para testar seus conhecimentos sobre Mortal Kombat?
